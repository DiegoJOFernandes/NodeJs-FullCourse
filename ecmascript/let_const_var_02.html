// Escopo é imprescindivel cada var tem um certo nível de permissão/acesso

// VAR // Temos 3 escopos // Escopo global -> Pode ser invocada dentro de qualquer outro escopo inferior(função e bloco) // Escopo de função -> function() {} // Escopo de bloco -> while() {} | if() {} | for() {} | switch() {}

// Hoisting -> Quando declaramos um operador VAR dentro de um bloco ou função ela sofre hoisting(elevação) para escopos superiores // var declarada em um bloco -> pode ser elevada para uma função ou uma variável global // var declarada em uma função -> pode ser elevada para uma variável global
